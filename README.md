<h1 align="center">🍰🥐 🍞 BAKERY SHOP WEBSITE 🍞🥐🍰</h1>
<br>
A fully responsive and dynamic Bakery Shop Website built using Bootstrap and APIs.  
This is a solo project focused on clean UI design, smooth user experience, and real-time data rendering using a custom JSON database.
<hr>

<h2>✨ Features</h2>

• 🎨 Modern & Attractive UI using Bootstrap  

• 📱 Fully Responsive Layout  

• 🔗 API Integration for Dynamic Data  

• 🍩 Bakery Product Showcase (Live Data)  

• ⚡ Fast & Interactive UI  

• 🗂️ Custom JSON Database  

• 👤 Solo Developer Project  

<hr>

<h2>🛠️ Tech Stack</h2>
<br>

✨ HTML5  
<br><br>
✨ CSS3  
<br><br>
✨ Bootstrap 5  
<br><br>
✨ JavaScript (Fetch API)  
<br><br>
✨ JSON (Custom Database)
